Webtoon Dashboard Website

2025 | Web Design, Dashboard

Proyek ini merupakan desain antarmuka dashboard untuk platform Webtoon. Tujuannya adalah membantu kreator melihat statistik pembaca, performa komik, dan grafik analitik secara real-time.

Desain ini dibuat dengan pendekatan clean layout dan kombinasi warna lembut agar mudah dibaca. Fokus utamanya adalah pengalaman pengguna yang sederhana namun informatif.

Setiap bagian dashboard menampilkan ringkasan data, seperti jumlah pembaca, episode populer, dan insight engagement. Desain menggunakan konsep grid agar data tersusun rapi dan mudah dipahami.
